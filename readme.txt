Automatize Daily Deals offers from affiliate networks 

== Description ==

**NOTE:** The *Synced Daily Deals* plugin requires that you have an active subscription on synced.io. If you are not already subscribed please check our list of prices here : [See pricing](https://synced.io/pricing)

The Synced Daily Deals provides access to our database of affiliate daily deals. We have aggregated over 5000 daily deals from over 100 merchants and 80 affiliate networks. All these daily deals are standardized in order to make this process easier for you.
The plugin uses woocommerce plugin to display the offers on the front blog, so you have to install and activate woocommerce plugin also with a frontstore theme.
 
At this point *Synced Daily Deals* plugin integrates with the following plugins:

* [WooCommerce](http://wordpress.org/plugins/woocommerce/)

The *Synced Daily Deals* plugin provides the interface to connect to the Synced.io API and perform the following tasks:

* Configures your API connection settings.
* Imports the data from your synced.io account
* Publishes / Unpublishes the daily deals offers
* Automatically removes the published offers which expire at next import
 
== Installation ==

This section describes how to install and configure the plugin:

1. Upload the `wpsynced` folder to the `/wp-content/plugins/` directory.
1. Activate the *Synced Daily Deals* plugin through the 'Plugins' menu in WordPress.
1. Enter your Synced.io endpoint, username and subscription id here: WordPress Admin Area > Synced > Synced
1. Import data from your synced account here: WordPress Admin Area > Synced > Merchants or Online Offers
1. Create your categories here: WordPress Admin Area > Products > Categories
1. Map the synced categories with your own created categories here: WordPress Admin Area > Synced > Categories > Assign
1. Publish your daily deals offers here: WordPress Admin Area > Synced > Online Offers
